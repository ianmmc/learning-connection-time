## Quick NCES ID Lookup

To find a district's NCES ID:
```bash
Grep(pattern="{District Name}.*{State}",
     path="data/processed/normalized/districts_2023_24_nces.csv",
     output_mode="content")
```

Example:
```bash
Grep(pattern="Memphis.*TN",
     path="data/processed/normalized/districts_2023_24_nces.csv",
     output_mode="content")
# Returns: 4700148,Memphis-Shelby County Schools,TN,110000,7500,2023-24,nces_ccd
```

**Note**: Memphis-Shelby County uses district ID **4700148** (not 4701980 as previously documented)
